# CodeClinic

CodeClinic is a hybrid coding-support platform that combines persistent AI conversations with live human collaboration in a single problem thread.

## Why I Built It

I built CodeClinic to bridge the gap between fast AI help and slower community feedback. Most tools force you to choose one or the other, but real learning and problem solving need both: immediate guidance plus human insight and review. This project brings them together so developers can get help quickly and still benefit from real peer collaboration.
